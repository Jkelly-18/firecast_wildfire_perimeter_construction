# FIRMS Data

Download VIIRS active fire data from: https://firms.modaps.eosdis.nasa.gov/download/

1. Select VIIRS (S-NPP, NOAA-20, NOAA-21)
2. Select California region or draw custom area
3. Select date range: 2021-2025
4. Download as shapefile (.zip)

Place the downloaded zip files in this folder. Expected files:
- DL_FIRE_J1V-C2_.zip (NOAA-20)
- DL_FIRE_J2V-C2_.zip (NOAA-21)
- DL_FIRE_SV-C2_.zip (Suomi NPP)
