# CAL FIRE Perimeter Data

Download California Fire Perimeters from: https://www.fire.ca.gov/what-we-do/fire-resource-assessment-program/fire-perimeters

1. Download "California Fire Perimeters (all)" shapefile
2. Extract contents to this folder

Expected files:
- California_Fire_Perimeters_(all).shp
- California_Fire_Perimeters_(all).dbf
- California_Fire_Perimeters_(all).prj
- California_Fire_Perimeters_(all).shx
