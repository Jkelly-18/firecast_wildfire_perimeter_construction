# CAL FIRE Perimeter Data

Download California Fire Perimeters from: https://gis.data.ca.gov/datasets/CALFIRE-Forestry::california-fire-perimeters-all/explore

1. Click the download icon
2. Select "Shapefile" as the format
3. Unzip the downloaded file and place contents directly in this folder

Expected files:
- California_Fire_Perimeters_(all).shp
- California_Fire_Perimeters_(all).dbf
- California_Fire_Perimeters_(all).prj
- California_Fire_Perimeters_(all).shx
